SKILL

MARK UP LANGUAGE

HTML

HTMLを使用して、webサイトの制作を行うことが可能です。コーディング全般で「自分以外が見てもわかりやすいコード」を意識して記述順やクラス名を統一し、見やすくなるように心がけています。

CSS

CSSを使用したwebサイトのスタイリングができます。 また、JavaScriptと組み合わせたり、keyframesやSVGを使ったアニメーションの実装も可能です。

SASS

Dart-sass(SCSS記法)を使用して、CSSを効率的に記述できます。mixinなどを利用しつつ、メンテナンスがしやすいようなコードになるように気を付けています。

PROGRAMMING LANGUAGE

Java Script

VanillaJSの基礎的な文法を学びました。現在も勉強中ですが、制作したWebサイトに動きを付けたり条件に合わせた制御を行うことができます。また、APIからデータを取得し、アプリの制作を行いました。

jQuery

jQueryを使用し、webサイトにアニメーション等の動きを付けることができます。

Node.js

Node.jsを利用した、環境構築やアプリケーション開発を勉強中です。また、Vue.jsなどのフレームワークも勉強しております。

DESIGN TOOLS

Photo Shop

画像の加工・修正が可能です。また切り抜き機能や簡単なパスを使用し、バナーやロゴのデザインを行うことができます。

Figma

Webサイトのワイヤーフレーム・デザインカンプの制作ができます。また、チームでの情報共有や制作も行うことができます。

Adobe XD

Webサイトのワイヤーフレーム・デザインカンプの制作ができます。

OTHER TOOLS

Git Hub

個人開発の範囲でのバージョン管理が可能です。管理にはターミナルのコマンドを使用しています。

Word Press

WordPressを使用したサイトを作成し、サーバーにアップロードが可能です。またオリジナルのテーマをPHPを使用して読み込み、WordPressの機能を実装をすることができます。

Chatwork
Quip

ChatworkやQuipを使用したコミュニケーションが可能です。文字だけの内容のため、要点が伝わるようにまとめたり、相手にとって気持ちの良い文章になるように意識しています。
